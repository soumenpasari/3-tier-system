Name of the project

Three Tier System (3-T-S)

The whole project is all for research purpose only and a work of college students for their final year engineering project.

The whole idea is to create or design a sytem or computer environment based on three tier architecture on which simple functions or program form client side will be processed by the secondary or main server i.e; the first server that interacts with the user and will return the results to the user and if the query or request from the user is of big data query then the main server or secondary server will request the the third system in this architecture i.e; the big data server and the that server will process the required query and return the result to the main server and the main server later will return the result to user or client side.

For big data functions we use the Hadoop for the query in the beginning, but the thing that we are researching on is to create a similar file system like HDFS that is used by Hadoop and after that we will analyse the time difference and all the differences or report that we get from the famous Hadoop and our own file system and will compare it later.

This project is a result of final year project at Netaji Subhash Engineering College by students of department Information Technology 4th Year under the guidance of Prof. Anirban Kundu.
